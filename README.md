# โครงสร้างองค์กร โรงพยาบาลชุมชนเทศบาลเมืองกำแพงเพชร

![GitHub top language](https://shields.io)
![Category](https://shields.io)

เอกสารระบุโครงสร้างองค์กรอย่างย่อและช่องทางการติดต่อของโรงพยาบาลชุมชนเทศบาลเมืองกำแพงเพชร (KPPMCH) สังกัดกองสาธารณสุขและสิ่งแวดล้อม เทศบาลเมืองกำแพงเพชร

---

## 👥 คณะผู้บริหารและส่วนงานปฏิบัติการ

### 1. ฝ่ายบริหารระดับสูง
*   **นายกเทศมนตรีเมืองกำแพงเพชร:** นายวุฒิชัย ศุภอรรถพานิช
*   **ผู้อำนวยการกองสาธารณสุขและสิ่งแวดล้อม:** (หัวหน้าหน่วยงานกำกับดูแลระดับกอง)
*   **หัวหน้าโรงพยาบาลชุมชนฯ:** นางปัทมา พวงมาลัย

### 2. ฝ่ายการแพทย์และบริการสาธารณสุข
*   **งานรักษาพยาบาล:** ทีมแพทย์ (นายแพทย์ / แพทย์หญิง) และพยาบาลวิชาชีพ
*   **งานส่งเสริมสุขภาพ:** ดูแอป้องกันเชิงรุกและการดูแลผู้ป่วยติดเตียงในชุมชน
*   **งานป้องกันและควบคุมโรค:** เฝ้าระวังและระงับการแพร่กระจายของโรคในเขตเทศบาล

### 3. ฝ่ายสนับสนุนและบริหารงานทั่วไป
*   **งานธุรการและบุคลากร:** งานสารบรรณและพนักงานจ้างทั่วไป
*   **แผนกการเงินและพัสดุ:** บริหารงบประมาณและการจัดซื้อจัดจ้าง
*   **เทคโนโลยีดิจิทัล:** ดูแลระบบสารสนเทศทางการแพทย์และฐานข้อมูลผู้ป่วย

*(หมายเหตุ: โครงสร้างรายละเอียดรายชื่อบุคคลอ้างอิงและอัปเดตตามโครงสร้างบุคลากรล่าสุด)*

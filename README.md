# 🏥 โรงพยาบาลชุมชนเทศบาลเมืองกำแพงเพชร (KPPMCH)

---

## 📜 ประวัติความเป็นมาของสถานพยาบาล
*   **พ.ศ. 2538 (จุดเริ่มต้น):** ก่อสร้างอาคารเดิมในชื่อ **“ศูนย์บริการสาธารณสุขเทศบาลเมืองกำแพงเพชร”** เป็นอาคารชั้นเดียวใต้ถุนสูง สังกัดกองการสาธารณสุขและสิ่งแวดล้อม เพื่อดูแลสุขอนามัยของประชาชนในชุมชน
*   **พ.ศ. 2558 (ยกระดับสู่โรงพยาบาลชุมชน):** จากนโยบายพัฒนาคุณภาพชีวิตและการเข้าถึงบริการทางการแพทย์อย่างเท่าเทียม จึงได้รับการอนุมัติเปลี่ยนชื่อและยกระดับอย่างเป็นทางการเป็น **"โรงพยาบาลชุมชนเทศบาลเมืองกำแพงเพชร"** เมื่อวันที่ 7 กันยายน พ.ศ. 2558
*   **ปัจจุบัน:** เป็นหน่วยบริการสุขภาพครบวงจรของท้องถิ่น ให้บริการครอบคลุมทั้งงานตรวจรักษาโรคทั่วไป ทันตกรรม ฝังเข็ม แพทย์แผนจีน แพทย์แผนไทย และงานเชิงรุกส่งเสริมสุขภาพชุมชน
